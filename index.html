Why would you want to use Scalapress? It's been proven in production - 100's of sites run on Scalapress. See the examples pages for more details. It's easily extensible. Anyone who can write Scala code can write a plugin. You don't need to deeply integrate with the platform as Scalapress will detect your plugins automatically on the classpath. It's high performance. There are plugins available for many different requirements - ecommerce, mailing lists, payment processors. It offers a highly flexible templating model that mean non developers can integrate with the features through easy to remember tags.
